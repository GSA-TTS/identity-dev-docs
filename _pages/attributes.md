# Available attributes

Here are the available attributes that can be requested when a user is validated.

| Attribute      | Default |
|----------------|---------|
| first_name     | Yes     |
| middle_name    | Yes     |
| last_name      | Yes     |
| address1       | Yes     |
| address2       | Yes     |
| city           | Yes     |
| state          | Yes     |
| zipcode        | Yes     |
| dob            | Yes     |
| ssn            | Yes     |
| phone          | Yes     |
| email          | No      |
